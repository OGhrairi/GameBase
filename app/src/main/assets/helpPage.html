GameBase Help Page

Overview

GameBase is an app that provides video game and game related information. The features include a news feed with a collation of recent video game related articles, a search function with filtering to find a game's description, and a top 50 game list, based on a combination of metrics.

News Feed

The news feed page retrieves a collated list of the 20 most recent video game related news articles, from a variety of sources. Pressing on any of the article cards will redirect you to the article, displaying within the app. From that screen, you can open the article in your web browser instead, should you wish.

Game Search

When opening the game search page, you will be presented by a text entry field and a series of buttons; one to perform the search based on an entered game title, one to add filters to the search, and one to clear the filters. Once a search has been performed, the results will be displayed on the screen. Tapping on any of these results will redirect you to a game information page, containing a description of the game and allowing you to share the game through external apps.

The search filter form contains a few fields:
Platform: to select a game platform, begin typing in the platform name. A dropdown will appear with autofill suggestions
Year: enter a date range for the search. Bear in mind that you do not need to set both

Top 50 Browse

The Top 50 page provides a list of the 50 highest rated games in the database, based on a combination of factors including average review, number of reviews and hype around the games. On selection of a game, you will be redirected to the same game information layout as the one in the search page.
